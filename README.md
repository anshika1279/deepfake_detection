# Deepfake Detection

A deep learning project for detecting deepfake images using advanced neural network architectures.

## Notebook
The output of the improved versions can be seen here:
- ([Deepfake_detection_improved_model_1.ipynb](https://colab.research.google.com/drive/1-tN3wUSvSgYyqMFiH5y3boGfOSehNZiE?usp=sharing)

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/anshika1279/deepfake_detection.git
   cd deepfake_detection
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the notebooks:
   ```bash
   jupyter notebook
   ```

## Technologies

- TensorFlow / Keras
- PyTorch
- OpenCV
- Vision Transformer (ViT)
- MobileNet / MobileNetV2
- scikit-learn
- NumPy & Pandas
